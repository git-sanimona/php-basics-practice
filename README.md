# php-basics-practice

## 概要

COACHTECH 教材 Tutorial 7-1「PHPの基礎 ハンズオン演習」で作成した成果物です。

PHPで商品価格の計算プログラムを作成しました。
商品の価格、数量、消費税率を使って合計金額を計算します。

## 使用技術

- PHP 8.x
  GitHub
  HTML
  Docker

## 学んだこと

- 計算プログラムで演算子と変数を使っての計算の書き方。

-文字列と変数の(.)を使った連結

-最初PHPをDockerでのcomposeを抜いてブラウザで確認しようとしてローカルホストに接続できなかったので慌てました。Webで調べてその後に教材を見直し自分のパソコンにPHPがインストールされておらずDockerでphp処理をしないといけないことが学べました。
